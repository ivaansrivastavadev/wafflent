# What is Wafflent?
Wafflent is a simple yet powerfull second brain that is not trying to replace your main note taking app but tries to co-work with it like in the interface you can add whats called a Waffle (a memory).
A Waffle Using a Audio file, Using a image (AI OCR/Tesseract-OCR) {LATER}, Using a video, etc... .

# Rules:
1. Always use the `tree` command when you do not know the structure
2. Always use the `/app/styles.css` file for css
3. Do not edit any of these files:
    - .gitignore
    - AGENTS.md
4. Do not make features marked as `[LATER]`
5. After doing any edit always run `./autotest.js` in the `localstore/tests/` directory
