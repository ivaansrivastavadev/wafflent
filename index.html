Features

Why Choose Wafflent?

Built with privacy and security at its core, Wafflent offers a new way to communicate.

End-to-End Encryption
Your messages are encrypted before leaving your device. Only you and your chat partner can read them.
Peer-to-Peer
Connect directly with others using WebRTC. No central servers means no data storage or surveillance.
Room Codes
Simple room codes make it easy to connect with friends. Share a code and start chatting instantly.
Real-time Communication
Instant message delivery with WebRTC data channels. Experience lightning-fast communication.
No Data Collection
We don't store your messages, track your activity, or collect personal data. Your privacy is guaranteed.
Cross-Platform
Works on any device with a modern web browser. No downloads or installations required.
How It Works

Get Started in Seconds

Wafflent makes secure communication simple and accessible to everyone.

Step 1
Create or Join Room
Enter a room code to join an existing chat, or create a new room and share the code with friends.
Step 2
Set Encryption Key
Choose a shared encryption key that both you and your chat partner know. This ensures only you can read the messages.
Step 3
Start Chatting
Once connected, start sending messages instantly. All communication is encrypted and peer-to-peer.
